# ByQuit
A simple python code supposed to be run on vps..  to close a specific position or all position at market price in Bybit. 

=== HOW TO USE ===

1. edit and check api in code 
2. make a venv: python3 -m venv venv
3. activate venv: source venv/bin/activate 
4. install pybit required: pip install pybit
5. run: python3 byquitsingle.py
