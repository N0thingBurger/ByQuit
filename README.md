# ByQuit
A simple python code supposed to be run on vps..  to close a specific position or all position at market price in Bybit. 

=== HOW TO USE ===
* edit and check api in code 

*make a venv: python3 -m venv venv
*activate venv: source venv/bin/activate 
*install pybit required: pip install pybit
*run: python3 byquitsingle.py
